This repository serves as a centralized collection of datasets and data analysis projects spanning various domains and industries. It is designed to host a diverse range of AI agent-driven analytical work, providing a clean and organized space for data-driven explorations and insights with human oversight.

## Repository Structure

Each project is self-contained within its own directory at the root, following a consistent structure:
- `<project_name>.csv`: Raw dataset.
- `<project_name>.ipynb`: Comprehensive Jupyter Notebook analysis.
- `<project_name>_report.md`: Detailed analytical report with visualizations.
- `visualizations/`: Folder containing all generated plots and charts.

## Current Projects

### 1. [Fungi Growth](./fungi_growth/)
An analysis of bioluminescent fungi growth factors in subterranean cave systems.
- **Key Focus:** Temperature, Humidity, and Spore Count correlations.

### 2. [Impact Study](./impact_study/)
A longitudinal analysis of Generative AI influence on university student academic performance and well-being.
- **Key Focus:** AI Literacy, GPA Impact, and Student Segmentation.

### 3. [Iris](./iris/)
A classical biological and statistical analysis of the Iris dataset.
- **Key Focus:** Morphological markers for species classification.
